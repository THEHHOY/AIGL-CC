# AIGL CC
AIGL CC Stands for a ImGui CC (ImGui is Immediate Mode Gui)

## What this does
Its a GUI/UI lib that allows you to make guis, but you pull the events manually and you
render stuff manually.

## a Code example.
Examples are at Examples folder

## Info about the library
You can make your own elements more on that on the docs.

## How to setup
To setup this you need to have your program and the uilib folder in the same directory.
You can also put the parent of uilib folder to package.path.
